Graph Panel
A panel that presents the visualisation of the current graph.
If the graph is analysed and it consists of multiple configurations, those are arranged into tabs. Please note that to interact with the graph you need to click (touch) once within the panel (a blue shaded border will appear). This denotes that page scrolling is temporarily disabled. To reverse this behaviour, just click (touch) once outside of the graph panel.

Instructions
Please also read the KEY information provided.
Every graph consists of nodes (circle, star or oval shapes) and connections (lines with arrows). The visualisation of the nodes can be changed using the Show node shapes / Show node labels option. The first time the graph is loaded, it is in its pre-analysed form. To analyse the graph, please select the 'Analyse Graph' option. To restore graph to its pre-analysed form, please select the 'Reset Graph' option. To interact with the graph, make sure a blue shaded border appears when you click (touch) once within the panel area. This will allow you to temporarily reposition the nodes, zoom in or out of the graph, move the graph in the space provided and also isolate first and second node connections.

To reposition a node, hold click on (touch) it and drag it to the desired position.
To zoom in/out scroll within the graph area (pinch-to-zoom on touchscreens, mouse scroll on desktop).
To move the graph around the space, hold click on (touch) the white space within the graph area and drag it to the desired position.
To reposition a node, hold click on (touch) it and drag it to the desired position.
To isolate first and second node connections, please click (touch) once the node you are interested in.

Options
You can find panel specific options on the panel's toolbar.
The general graph options can be found on the 'Information' right hand side menu.
The general graph options can be found on the top main toolbar (right hand side).

Available panel options:

KEY Opens the graph's visualisation KEY in a dialogue.
fullscreen Opens the graph in a dialogue.
camera Takes a screenshot of the current graph and saves it as an image.
stars / font_download Change visualisation of the graph. Nodes appear as ellipse with text inside / Nodes appear as shapes with text at the bottom.
help Opens help dialogue.

Available graph options:

Analyse Graph Uses our CCTool algorithmic approach to analyse the graph and return back any control nodes and stems.
Edit Graph Allows you to make modifications to the current graph. That includes details of the graph, node properties, node connections, etc.
Duplicate Graph Creates a copy of the current graph.
Reset Graph Discard any analysis of the current graph..
Remove Graph Deletes the current graph from the account. Please notice that this action cannot be reverted.
Compare Graph Offers the ability to compare the current graph with another existing graph on the logged-in account.
New Graph Offers a tool to add a new graph to the account.

Definitions
Node Nodes are the “things” that a network connects together. They can be factors, objects, or concepts.
Edge An edge is a link between two (2) nodes in a network. Here the edges represent causal influences.
Control Node Control nodes are nodes that because of their network position can be shown mathematically to be able to influence a large proportion of the network through a direct causal chain. A set of control nodes together can exert control over the whole network.

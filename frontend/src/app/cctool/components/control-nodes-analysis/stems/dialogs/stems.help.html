Stems Panel
A panel that presents the stems generated from the active configuration of the current graph. Please note that the panel offers both vertical and horizontal scolling for very long stems.

Options
You can find panel specific options on the panel's toolbar.
The general graph options can be found on the 'Information' right hand side menu.
The general graph options can be found on the top main toolbar (right hand side).

Available panel options:

KEY Opens the graph's visualisation KEY in a dialogue.
fullscreen Opens the graph in a dialogue.
stars / font_download Change visualisation of the graph. Nodes appear as ellipse with text inside / Nodes appear as shapes with text at the bottom.
help Opens help dialogue.

Available graph options:

Analyse Graph Uses our CCTool algorithmic approach to analyse the graph and return back any control nodes and stems.
Edit Graph Allows you to make modifications to the current graph. That includes details of the graph, node properties, node connections, etc.
Duplicate Graph Creates a copy of the current graph.
Reset Graph Discard any analysis of the current graph..
Remove Graph Deletes the current graph from the account. Please notice that this action cannot be reverted.
Compare Graph Offers the ability to compare the current graph with another existing graph on the logged-in account.
New Graph Offers a tool to add a new graph to the account.

Definitions
Node Nodes are the “things” that a network connects together. They can be factors, objects, or concepts.
Edge An edge is a link between two (2) nodes in a network. Here the edges represent causal influences.
Control Node Control nodes are nodes that because of their network position can be shown mathematically to be able to influence a large proportion of the network through a direct causal chain. A set of control nodes together can exert control over the whole network.
